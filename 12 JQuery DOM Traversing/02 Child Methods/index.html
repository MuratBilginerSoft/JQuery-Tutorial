Çocuk Yöntemleri

jQuery ile DOM elementlerinin alt elementlerine erişme yöntemleri

children() Metodu

Seçili elementin doğrudan alt elementlerini seçer. İsteğe bağlı olarak seçici (selector) kullanılabilir.

Çocuk 1

Çocuk 2

Alt çocuk 

// Tüm doğrudan çocukları seç
$(".parent").children().css("border", "2px solid blue");

// Sadece .special sınıfına sahip çocukları seç
$(".parent").children(".special").css("background", "yellow");

find() Metodu

Seçili elementin altındaki tüm elementler içinde arama yapar. Belirtilen seçiciye uyan tüm alt elementleri bulur.

Paragraf 1

Paragraf 2

Önemli metin 

// Tüm span elementlerini bul
$(".container-element").find("span").css("color", "red");

// .highlight sınıfına sahip elementleri bul
$(".container-element").find(".highlight").css("background", "yellow");
