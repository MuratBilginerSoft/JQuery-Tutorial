Filtreleme Yöntemleri

jQuery ile DOM elementlerini filtreleme yöntemleri

first() ve last() Metodları

first() seçili elementler kümesindeki ilk elementi, last() son elementi seçer.

Element 1
Element 2
Element 3
Element 4
Element 5

// İlk elementi seç
$(".item").first().css("background", "lightblue");

// Son elementi seç
$(".item").last().css("background", "lightgreen");

eq() Metodu

Seçili elementler kümesinden belirtilen indeksteki elementi seçer.

İndeks 0
İndeks 1
İndeks 2
İndeks 3
İndeks 4

// İndeks 2'deki elementi seç
$(".item").eq(2).css("background", "yellow");

// Son elementin bir öncesini seç (negatif indeks)
$(".item").eq(-2).css("border", "2px solid red");

filter() Metodu

Seçili elementler kümesini belirtilen kritere göre filtreler.

Normal Element
Özel Element
Normal Element
Özel Element
Normal Element

// .special sınıfına sahip elementleri filtrele
$(".item").filter(".special").css("background", "lightpink");

// Fonksiyon ile filtreleme
$(".item").filter(function(index) {
    return index % 2 === 0;
}).css("border", "2px dashed blue");

not() Metodu

Seçili elementler kümesinden belirtilen kritere uyan elementleri çıkarır.

Hariç Tutulacak
Dahil Edilecek
Hariç Tutulacak
Dahil Edilecek
Hariç Tutulacak

// .exclude sınıfına sahip olmayan elementleri seç
$(".item").not(".exclude").css("background", "lightgreen");

// Fonksiyon ile hariç tutma
$(".item").not(function(index) {
    return index % 2 === 0;
}).css("border", "2px solid purple");

Filtreleme Metodları Karşılaştırması

Metod Kullanım Amacı Parametre Türü Örnek Kullanım
first() İlk elementi seçme - $("div").first()
last() Son elementi seçme - $("div").last()
eq() İndekse göre seçim Number $("div").eq(2)
filter() Kritere göre filtreleme Selector/Function $("div").filter(".class")
not() Kritere uyanları hariç tutma Selector/Function $("div").not(".class")
