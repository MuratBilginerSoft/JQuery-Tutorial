jQuery DOM Traversing

DOM elementleri arasında gezinme ve element seçme yöntemleri

Ebeveyn Yöntemleri

parent(), parents(), parentsUntil()

Çocuk Yöntemleri

children(), find()

Kardeş Yöntemleri

siblings(), next(), prev(), nextAll(), prevAll()

Filtreleme Yöntemleri

first(), last(), eq(), filter(), not()

Hızlı Başvuru

Yukarı Doğru Traversing

  • parent() - Doğrudan üst element
  • parents() - Tüm üst elementler
  • parentsUntil() - Belirli bir elemente kadar olan üst elementler

Aşağı Doğru Traversing

  • children() - Doğrudan alt elementler
  • find() - Tüm alt elementler

Yatay Traversing

  • siblings() - Tüm kardeş elementler
  • next()/prev() - Sonraki/önceki element
  • nextAll()/prevAll() - Tüm sonraki/önceki elementler

Filtreleme

  • first()/last() - İlk/son element
  • eq() - Belirli indeksteki element
  • filter()/not() - Koşula uyan/uymayan elementler
