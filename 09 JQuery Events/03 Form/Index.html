Form Olayları

jQuery ile form olaylarını nasıl yönetebileceğinizi öğrenin

Focus ve Blur Events

Durum: Input alanı beklemede 
$('#element')
    .focus(function() { /* Odaklanıldığında */ })
    .blur(function() { /* Odak kaybedildiğinde */ });

Change Event

Seçilen renk burada görünecek 
$('#element').change(function() {
    let selectedValue = $(this).val();
    // Değer değiştiğinde çalışır
});

Submit Event

Form henüz gönderilmedi 
$('#form').submit(function(event) {
    event.preventDefault();
    // Form gönderildiğinde çalışır
});

Input Events

Karakter sayısı: 0
Kelime sayısı: 0
$('#element').on('input', function() {
    // Her karakter girildiğinde çalışır
});

Form Doğrulama

$('input').on('input', function() {
    validateInput($(this));
});
