Document/Window Olayları

jQuery ile document ve window olaylarını nasıl yönetebileceğinizi öğrenin

Document Ready Event

Sayfa yükleme durumu 
$(document).ready(function() {
    // Sayfa yüklendiğinde çalışır
});

Scroll Event

Scroll Pozisyonu

Scroll pozisyonu: 0px
$(window).scroll(function() {
    let scrollPos = $(window).scrollTop();
    // Scroll pozisyonu değiştiğinde çalışır
});

Resize Event

Pencere boyutu: - 
$(window).resize(function() {
    let width = $(window).width();
    let height = $(window).height();
    // Pencere boyutu değiştiğinde çalışır
});

Load/Unload Events

Görsel yükleniyor... 
$(window).on('load', function() {
    // Tüm kaynaklar yüklendiğinde çalışır
});

$(window).on('beforeunload', function() {
    // Sayfa kapatılmadan önce çalışır
});

Error Event

Hata kayıtları burada görünecek 
$(element).on('error', function(event) {
    // Hata oluştuğunda çalışır
    console.error('Bir hata oluştu:', event);
});

Visibility Change

Sayfa görünür durumda
Başka bir sekmeye geçerek deneyin 
$(document).on('visibilitychange', function() {
    let state = document.visibilityState;
    // Sayfa görünürlüğü değiştiğinde çalışır
});
