Klavye Olayları

jQuery ile klavye olaylarını nasıl yönetebileceğinizi öğrenin

Keypress Event

Henüz bir tuşa basılmadı 
$('#element').keypress(function(event) {
    // Tuşa basıldığında çalışır
    let key = String.fromCharCode(event.which);
});

Keydown/Keyup Events

Tuş durumu: Beklemede 
$('#element')
    .keydown(function(event) { /* Tuşa basıldığında */ })
    .keyup(function(event) { /* Tuş bırakıldığında */ });

Özel Tuşlar

Basılan özel tuşlar: - 
$('#element').keydown(function(event) {
    if (event.ctrlKey) { /* Ctrl basılı */ }
    if (event.altKey) { /* Alt basılı */ }
    if (event.shiftKey) { /* Shift basılı */ }
});

Oyun Kontrolü Örneği

◉

Yön tuşlarını kullanarak karakteri hareket ettirin

  • ↑ : Yukarı
  • ↓ : Aşağı
  • ← : Sol
  • → : Sağ
$(document).keydown(function(event) {
    switch(event.which) {
        case 37: // sol
        case 38: // yukarı
        case 39: // sağ
        case 40: // aşağı
    }
});

Tuş Kombinasyonları

Aşağıdaki kombinasyonları deneyin:

  • Ctrl + S
  • Ctrl + A
  • Ctrl + Z
Son kullanılan kombinasyon: - 
$(document).keydown(function(event) {
    if (event.ctrlKey && event.key === 's') {
        event.preventDefault();
        // Ctrl + S işlemi
    }
});
