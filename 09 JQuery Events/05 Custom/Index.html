Özel Olaylar

jQuery ile kendi özel olaylarınızı nasıl oluşturabileceğinizi öğrenin

Temel Özel Olay

Olay bekleniyor... 
// Özel olayı tanımla
$('#element').on('customEvent', function() {
    // Olay tetiklendiğinde çalışır
});

// Özel olayı tetikle
$('#element').trigger('customEvent');

Veri İçeren Özel Olay

Veri bekleniyor... 
// Veri içeren olayı tetikle
$('#element').trigger('customEvent', {
    message: 'Merhaba!',
    timestamp: new Date()
});

Olay İsim Alanları

Tetiklenen olaylar burada listelenecek... 
// İsim alanı ile olay tanımla
$('#element').on('click.demo', function() {
    // Sadece .demo isim alanındaki click olayı
});

// Belirli isim alanındaki olayları tetikle
$('#element').trigger('click.demo');
