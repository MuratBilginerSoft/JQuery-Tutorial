Mouse Olayları

jQuery ile fare olaylarını nasıl yönetebileceğinizi öğrenin

Click Events

Bana tıkla!
Tıklama sayısı: 0 
$('#element').click(function() {
    // Tıklama işlemi
});

Double Click Events

Bana çift tıkla!
Henüz çift tıklama yapılmadı 
$('#element').dblclick(function() {
    // Çift tıklama işlemi
});

Hover Events

Üzerime gel!
Mouse durumu: Dışarıda 
$('#element').hover(
    function() { // Mouse üzerine gelince },
    function() { // Mouse ayrılınca }
);

Mouse Position

Bu alanda fareyi hareket ettir
X: 0, Y: 0 
$('#element').mousemove(function(event) {
    let x = event.pageX;
    let y = event.pageY;
});
