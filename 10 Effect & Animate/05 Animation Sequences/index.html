Animasyon Sekansları

jQuery'de ardışık ve paralel animasyonları öğrenin

Temel Sekans

Sekans Demo 
$("#element")
    .fadeOut("slow")
    .fadeIn("slow")
    .animate({left: "200px"});

Callback ile Sekans

Callback Demo 
$("#element").fadeOut("slow", function() {
    $(this).fadeIn("slow", function() {
        $(this).animate({left: "200px"});
    });
});

Promise ile Sekans

Promise Demo 
$("#element").promise().then(function() {
    return $(this).fadeOut("slow").promise();
}).then(function() {
    return $(this).fadeIn("slow").promise();
});
