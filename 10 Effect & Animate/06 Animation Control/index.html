Animasyon Kontrolü

jQuery'de animasyonları kontrol etmeyi öğrenin

Stop Metodu

Stop Demo 
// Sadece mevcut animasyonu durdur
$("#element").stop();

// Tüm animasyonları durdur
$("#element").stop(true);

// Son konuma atla
$("#element").stop(true, true);

Delay Metodu

Delay Demo 
$("#element")
    .fadeOut()
    .delay(1000)
    .fadeIn();

Finish Metodu

Finish Demo 
// Tüm animasyonları hemen tamamla
$("#element").finish();

Queue ve Dequeue

Queue Demo 
// Kuyruğa fonksiyon ekle
$("#element").queue(function() {
    $(this).addClass("active");
    $(this).dequeue();
});
