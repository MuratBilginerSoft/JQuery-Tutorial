Özel Animasyonlar

jQuery'nin animate() metodunu kullanarak özel animasyonlar oluşturun

Temel Animasyon

Hareket Demo 
$("#element").animate({
    left: "200px"
}, "slow");

Çoklu Özellik Animasyonu

Multi Demo 
$("#element").animate({
    left: "200px",
    width: "300px",
    opacity: 0.5
}, "slow");

Göreceli Değerler

Göreceli Demo 
$("#element").animate({
    width: "+=50px",
    height: "+=50px"
}, "slow");

Easing Fonksiyonları

Easing Demo 
$("#element").animate({
    left: "200px"
}, {
    duration: "slow",
    easing: "swing"
});

Step Fonksiyonu

Step Demo 
$("#element").animate({
    left: "200px"
}, {
    duration: 2000,
    step: function(now, fx) {
        // Her adımda çalışır
        console.log(now);
    }
});
