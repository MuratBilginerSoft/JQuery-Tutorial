jQuery Seçiciler Galerisi

jQuery'nin güçlü seçici özelliklerini keşfedin

Tag Seçiciler

HTML etiketlerini seçme yöntemleri

$('p'), $('div'), $('span')

ID Seçiciler

Benzersiz ID'ye sahip öğeleri seçme

$('#elementId')

Class Seçiciler

CSS sınıflarına göre öğe seçme

$('.className')

Özellik Seçiciler

HTML özelliklerine göre seçme

$('[type="text"]')

Alt Öğe Seçiciler

Alt öğeleri seçme yöntemleri

$('ul > li')

Üst Öğe Seçiciler

Üst öğeleri seçme yöntemleri

$('element').parent()

İçerik Seçiciler

Metin içeriğine göre seçme

$('p:contains("text")')

Gelişmiş Seçiciler

İleri düzey seçme teknikleri

$('li:first'), $('li:even')

Zincirleme Seçiciler

Seçicileri birleştirme teknikleri

$('.class1, .class2')
